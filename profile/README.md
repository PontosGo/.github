# PontosGo

![PontosGo](assets/logo.png)

Plataforma SaaS de fidelidade baseada em XML fiscal para comerciantes e consumidores.

## Repositorios

- `pontosgo-platform`: arquitetura, playbook, workspaces e customizacoes de Copilot
- `pontosgo-backend`: API principal, contratos e processamento de XML
- `pontosgo-admin`: painel interno da operacao PontosGo
- `pontosgo-comerciante`: painel web do estabelecimento
- `pontosgo-cliente`: aplicacao web do cliente final
- `pontosgo-app`: aplicativo mobile do cliente final
- `pontosgo-coletor`: coletor de XML fiscal no PDV
- `pontosgo-infra`: infraestrutura local compartilhada

## Fluxo De Branches

- `feature/*` -> `dev`
- `dev` -> `main`
